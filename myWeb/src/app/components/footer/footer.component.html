Website information

This website was constructed as a SPA (Single Page Application) using Angular v5. It is currently being hosted on Google's Firebase Hosting servers. The choice to host it there was made because, some of the data that is being render on this page is dynamically being pulled from Firebase's Realtime Database. Below is a list of tools that are were used to create this web application. As well at the bottom there is an admin login page where I can logon and modify the dynamic content.

  • Tools used:
    • Angular v5
    • VS Code
    • Google Chrome Dev Tool
    • Google Firebase service
  • Admin Login
